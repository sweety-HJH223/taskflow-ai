# ⚡ TaskFlow AI

A modern SaaS landing page for an AI-powered freelance workflow automation tool. Built as a freelance portfolio demo project targeting global clients.

## 🔗 Live Demo
[taskflow-ai.vercel.app](https://taskflow-ai.vercel.app)


## ✨ Features
- Editorial dark design with lime green accents
- Animated marquee ticker strip
- Asymmetric hero with AI task assistant card
- Problem section with stat cards
- Bento grid features layout
- Numbered how it works steps
- "Also Built By SweetyCodes" section linking to real AI tools
- FAQ accordion with smooth animations
- Waitlist form with Formspree integration
- Minimal single-line footer
- Mobile responsive with hamburger menu

## 🛠️ Tech Stack
- Next.js 16
- Tailwind CSS
- TypeScript
- Formspree (waitlist form backend)
- Vercel (deployment)

## 🚀 Getting Started
```bash
git clone https://github.com/sweety-HJH223/taskflow-ai.git
cd taskflow-ai
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📋 How the Waitlist Works
1. Visitor enters email in waitlist form
2. Data sent to Formspree API
3. Owner receives email notification
4. Success message shown to visitor
